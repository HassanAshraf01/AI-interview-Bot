# 🎤 AI Voice-Based Interview Bot

## 📌 Project Description

The AI Voice-Based Interview Bot is a web application designed to simulate a real interview environment. It interacts with users through voice-based questions and responses, helping candidates practice and improve their interview skills.

This project focuses on creating a realistic interview experience with proper timing, question flow, and user interaction.

---

## 🚀 Features

* 🎙️ Voice-based interview interaction (input & output)
* ⏱️ Real-time interview simulation with delays
* 👤 Separate interview modes (Male/Female interviewer UI)
* 🔐 User authentication (Login/Signup)
* 📊 Structured interview question flow
* 🌐 Responsive frontend design

---

## 🛠️ Tech Stack

* **Backend:** Django
* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Database:** SQLite
* **Other Tools:** Web Speech API (for voice handling)

---

## ⚙️ How It Works

1. User logs into the system
2. Selects interview type
3. System asks questions with realistic delay
4. User responds using voice/text
5. Flow continues like a real interview
